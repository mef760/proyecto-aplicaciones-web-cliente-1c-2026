¿Qué construí en esta clase?
Un sitio de e-commerce para una tienda de tecnología y gaming llamada "TechStore" con:

Página principal (index.html) con ofertas y categorías
Catálogo de productos (productos.html) con 23 productos en 6 categorías
Página Sobre Nosotros con historia de la empresa
Carrito de compras (carrito.html) con tabla de productos, opciones de envío y pago
Formulario de contacto y alta de productos
Buscador sin JavaScript usando checkbox oculto
Grilla responsive con CSS Grid
Explicación generada por OpenCode
El sitio fue construido utilizando HTML5 semántico y CSS3 moderno, implementando:
CSS Grid para las grillas de productos (1/2/3 columnas responsive)
Truco del checkbox oculto para el buscador sin JS
Tarjetas de productos con estructura de imagen, info y descripción desplegable en hover
Estética con bordes rectos (border-radius: 0px) y colores definidos por variables CSS
¿Qué cambié o corregí de lo que generó la IA?
El buscador inicialmenteabiía hacia la izquierda y tapaba el nav → corregido para abrir hacia abajo
Las ofertas estaban separadas en una sección propia → integradas dentro de cada categoría
La descripción de las tarjetas tapaba el precio → corregido para que se despliegue solo sobre la imagen (parte superior)
Los precios de ofertas ahora están en línea (uno al lado del otro) en lugar de uno debajo del otro
El fondo de la descripción era negro → cambiado a gris claro (#f5f5f5)
Referencia de imágenes agregadas con nombres cortos (proc-01.jpg, gpu-02.jpg, etc.)
